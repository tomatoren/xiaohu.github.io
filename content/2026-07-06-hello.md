# 开博第一篇

这是用**零依赖纯静态站点**写下的第一篇文章。整套方案不装任何 npm 包、不需要数据库，写完 Markdown 提交就自动部署。

## 为什么这样选

- **费用低**：托管在 GitHub Pages，完全免费。
- **安全**：纯静态文件，没有服务器可被入侵。
- **发布方便**：写 `.md` → `git push` → 自动上线。
- **好接入平台**：自带 RSS，阅读器和各内容平台都能订阅。

> 简单，才可持续。

## 怎么写新文章

只要在 `content/` 目录新建一个 `.md` 文件，头部加一段 frontmatter：

```md
---
title: 文章标题
date: 2026-07-06
tags: [标签1, 标签2]
description: 一句话摘要
---

正文用 Markdown 写就好。
```

然后提交，几秒钟后就出现在首页和 RSS 里了。

## 以后怎么分发

- 用阅读器订阅 `feed.xml` 即可第一时间收到更新。
- 用 **wechatsync** 这类浏览器插件，可以把文章一键同步到公众号、知乎、小红书。
- 想更自动？将来可以加个小脚本，调各平台开放 API 批量转发。
